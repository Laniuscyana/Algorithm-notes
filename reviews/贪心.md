# 贪心
## LC.
> https://leetcode.cn/problems/generate-a-string-with-characters-that-have-odd-counts/

```java
class Solution {
    public String generateTheString(int n) {
        char[] ans=new char[n];
        String anss="";
        if(n%2==1){
            for(int i=0;i<n;i++){
                ans[i]='a';
                anss +=ans[i];
            }
        }else{
            for(int i=0;i<n-1;i++){
                ans[i]='a';
                anss +=ans[i];
            }
            ans[n-1]='b';
            anss +=ans[n-1];
        }
        return anss;
    }
}
```
